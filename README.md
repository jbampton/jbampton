## Random Maths

The relationship between sine and cosine is famously shown with the Pythagorean identity:

```math
\sin^2\theta + \cos^2\theta = 1
```

Absolute value or modulus of a complex number $a + bi$ is 

```math
\\abs(a + bi)
```
